## This is a mobile app that represent Pokedex

### This app consists of 3 screens
- Pokemon list
- Pokemon detail
- User favorite pokemon

### Tech stack
- React-Native with Expo
- Typescript
- Axios
- Zodios
- React Query
- React native MMKV
- FlashList
- Expo Image
